Este trabalho aborda a criação e manipulação de um banco de dados
denominado “Loja”, utilizando a linguagem SQL. As principais etapas incluem a
criação da estrutura do banco de dados, desenvolvimento da estrutura do banco de
dados com base em um diagrama entidade-relacionamento (DER) pré-definido,
utilização do MySQL Server e do software MySQL Workbench para criar a base
de dados.
Em seguida será realizada a etapa de inserção de registros, populando as
tabelas do banco de dados com registros fictícios. O script conterá os comandos de
manipulação (DML) necessários para inserir pelo menos três registros em cada
tabela.
O trabalho se encerra com a criação da visão “ContasNaoPagas”, que retornará
todas as contas pendentes de pagamento contendo o ID da conta a receber, nome
e CPF do cliente associado à conta, data de vencimento da conta e o valor da
conta.
